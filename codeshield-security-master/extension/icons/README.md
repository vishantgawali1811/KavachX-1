# CodeShield Extension Icons

This directory contains the icons for the CodeShield browser extension.

## Required Icons:
- icon16.png - 16x16 pixels
- icon32.png - 32x32 pixels  
- icon48.png - 48x48 pixels
- icon128.png - 128x128 pixels

## Icon Design:
The icons should feature a shield (🛡️) symbol representing security and protection.
Color scheme should use:
- Primary: #667eea (blue)
- Secondary: #764ba2 (purple)
- Success: #28a745 (green)
- Warning: #ffc107 (yellow)
- Danger: #dc3545 (red)

## Temporary Note:
For development/testing, you can use placeholder icons or download from:
https://github.com/ishwari05/codeshield-security/tree/main/extension/icons

## Icon States:
- Normal: Blue shield
- Active/Protected: Green shield with checkmark
- Warning: Yellow shield with exclamation
- Danger: Red shield with lock
